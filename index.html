Frontend Developer

Mijn naam is Zekkie Berkane, een CMD afstudeer student aan de HVA. Ik ben opzoek naar een stage plek waar ik mijn skills als frontend developer kan uitbreiden. Voor ik mijn studie begon aan de HVA heb ik de opleiding interactief vormgever gedaan aan de GLU. Momenteel ben ik werkzaam als bezorger bij de persgroep.

Verder hou ik van lezen, gamen, programmeren en leer ik graag nieuwe dingen.

Mijn werk

Webapp from scratch

Webapp from scratch was een opdracht die wij tijdens de minor gedaan hebben. De bedoeling was dat wij zonder libraries(met een uitzondering op microlibs) een webapp zouden gaan maken.

Voor deze opdracht had ik een site gemaakt die gebruik maakte van een nieuws API. De gebruiker krijgt een overzicht van het laatste nieuws die in de wereld gaande is.

Tijdens deze opdracht had ik mijzelf uitgedaagd om een eigen templating en een routing engine te gaan maken. Dus de code is in zijn geheel zelf door mij geschreven.

Functional Programming

Tijdens functional programming hebben we leren werken met functional patterns. Hiervoor hebben wij data van de RDW gebruikt. Deze hebben wij opgeschoond en getransformeerd naar bruikbare data.

Met behulp van d3.js hebben we deze data omgezet naar een interactieve grafiek. Met mijn grafiek kan je zien hoeveel parkeermeters zijn bij gekomen en verdwenen door de jaren heen.

Linernote database

Linernote was een klant tijdens het eindproject van de minor web development. De klant wilde een applicatie die allerlei informatie toont over een muzikant. Bijvoorbeeld instagram posts, youtube uploads en wiki data.

Voor de opdracht heb ik een backend gemaakt die api calls doet naar youtube, spotify en wikipedia. Omdat instagram geen api endpoints meer heeft waarbij je posts van gebruikers kan ophalen. Had ik een scraper gemaakt die posts ophaalt. Deze data kon de frontend ophalen.
